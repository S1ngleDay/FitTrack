# 🎮 FitTrack Demo Mode

## Включение/Выключение демо-режима

Демо-режим предназначен для создания скрин-шотов презентации и тестирования приложения с реальными данными.

### **Как включить/выключить:**

**Открыть файл: `src/utils/demoData.js`**

```javascript
export const DEMO_MODE = true;  // 🎮 Переключатель режима демо
```

- **`DEMO_MODE = true`** — Демо-режим **ВКЛЮЧЕН** (приложение загружается с тестовыми данными)
- **`DEMO_MODE = false`** — Демо-режим **ВЫКЛЮЧЕН** (приложение работает нормально)

---

## Что загружается в демо-режиме?

### 📊 **20 тренировок** за 3 месяца
- Обычные тренировки: бег 🏃, велосипед 🚴, ходьба 🚶, кардио 💪
- Силовые тренировки с полным описанием упражнений
- GPS маршруты для бега и велосипеда
- Реалистичные метрики (время, расстояние, калории, шаги)

### 📋 **3 плана тренировок**
- Марафонская подготовка (бег)
- Постройка мышц (силовая, PPL сплит)
- Похудение через кардио

### 👤 **Демо-пользователь**
- **Имя:** Петр Иванов
- **Вес:** 75 кг
- **Рост:** 182 см
- **Возраст:** 28 лет
- **Цели:** 2000 ккал/день, 10000 шагов/день

### ⚙️ **Демо-настройки**
- Тема: Темная (`isDark: true`)
- Уведомления: Включены
- Язык: Русский (ru)

## 📈 Статистика в демо-режиме

При включенном демо-режиме экран статистики покажет:

1. **График активности за последний месяц** (BarChart)
2. **Распределение типов тренировок** (PieChart)
3. **Нагрузка на мышцы** (распределение по группам)
4. **Общие统计:**
   - Всего тренировок: 20
   - Общее время: ~900 минут
   - Общие калории: ~8000 ккал
   - Общее расстояние: ~150 км
   - Общий объем для силовых: ~5000 кг

---

## 🔄 Как это работает?

1. **При старте приложения** — хук `useDemoDataInit()` в `App.js` проверяет флаг `DEMO_MODE`
2. **Если `DEMO_MODE = true`** — загружает демо-данные в Zustand store
3. **Данные сохраняются** в локальное хранилище (AsyncStorage)
4. **При следующем запуске** приложения данные будут уже загружены

---

## 🎥 Рекомендации для скрин-шотов

Для получения хороших скрин-шотов презентации:

1. **Включите демо-режим** (`DEMO_MODE = true`)
2. **Перезагрузите приложение** (полный перезапуск, не горячий перезагруз)
3. **Перейдите на экран Статистики** для красивых графиков
4. **На главном экране** будут видны активности сегодняшнего дня
5. **На экране Тренировок** отобразятся все 20 тренировок

---

## ⚠️ Важно!

- **Демо-данные перезаписывают** существующие данные пользователя!
- Перед отключением демо-режима **закончите работу с приложением**
- Заф разработки: всегда запускайте приложение на `DEMO_MODE = true`, чтобы иметь тестовые данные
- Перед релизом: все**ОБЯЗАТЕЛЬНО** установите `DEMO_MODE = false`
