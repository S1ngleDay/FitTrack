---

## 🎯 Что это?

Система для быстрой загрузки реалистичных тестовых данных в приложение FitTrack одной строкой кода.

**Использование:** 30 секунд на включение/выключение.

---

## 🚀 БЫСТРЫЙ СТАРТ

### **Включить демо:**
```bash
1. Откройте: src/utils/demoData.js
2. Строка 3:  export const DEMO_MODE = true;  ✅
3. Перезагрузите приложение
4. ГОТОВО! 🎉
```

### **Выключить демо:**
```bash
1. Откройте: src/utils/demoData.js
2. Строка 3:  export const DEMO_MODE = false;  ❌
3. Перезагрузите приложение
```

## 📞 Часто задаваемые вопросы

**Q: Как добавить свои данные?**
A: Отредактируйте массивы в `src/utils/demoData.js`

**Q: Почему данные не загружаются?**
A: Используйте полный перезагруз (не горячий). Очистите консоль.

**Q: Можно ли переключать на лету?**
A: Нет, нужно перезагрузить приложение.

**Q: Влияет ли на производительность?**
A: Нет, только первый запуск медленнее на ~100ms.
