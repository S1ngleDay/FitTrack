## 🚀 КАК ИСПОЛЬЗОВАТЬ?

### **В одну минуту:**
1. Откройте `src/utils/demoData.js`
2. Убедитесь: `export const DEMO_MODE = true;`
3. Перезагрузите приложение
4. ✨ Все! Приложение полно реалистичных данных

### **Перед финальной сдачей:**
1. Откройте `src/utils/demoData.js`
2. Измените: `export const DEMO_MODE = false;`
3. Перезагрузите приложение
4. Приложение работает нормально (без демо-данных)
