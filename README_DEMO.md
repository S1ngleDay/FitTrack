## ⚡ БЫСТРАЯ ИНСТРУКЦИЯ

### **Включить демо:**
1. Откройте `src/utils/demoData.js`
2. Найдите `export const DEMO_MODE = true;`
3. Перезагрузите приложение

### **Выключить демо:**
1. Откройте `src/utils/demoData.js`
2. Измените на `export const DEMO_MODE = false;`
3. Перезагрузите приложение

## 📞 ЧАСТО ЗАДАВАЕМЫЕ ВОПРОСЫ

**Q: Данные не загружаются?**
A: Используйте полный перезагруз (не горячий). Очистите консоль.

**Q: Как добавить свои данные?**
A: Отредактируйте массив `demoDemoWorkouts` в `demoData.js`

**Q: Влияет ли на размер приложения?**
A: Нет, только в памяти разработчика, легко удаляется.

**Q: Проблемы при выключении?**
A: Установите `DEMO_MODE = false` и перезагрузитесь.
