# ⚡ ОДНОСТРАНИЧНЫЙ ГАЙД: Включить демо за 30 секунд

## 🎯 Что вам нужно?

Вы хотите показать приложение с красивыми данными на презентации.

## ✅ Как это сделать?

### **Шаг 1 (10 сек)**
Откройте файл:
```
src/utils/demoData.js
```

### **Шаг 2 (5 сек)**
Найдите строку 3:
```javascript
export const DEMO_MODE = true;  // ← Должно быть true (так у вас и есть)
```

### **Шаг 3 (10 сек)**
Перезагрузите приложение (полный перезагруз, не горячий)

### **Готово! ✨**

## ❌ После презентации (Выключить!)

Ещё раз откройте `src/utils/demoData.js` и измените:
```javascript
export const DEMO_MODE = false;  // ← Измените на false перед финальной сдачей
```

---

## 📞 Если что-то не работает

| Проблема | Решение |
|----------|---------|
| Данные не загружаются | Полный перезагруз (не горячий refresh) |
| Консоль пуста | Очистите консоль и перезагрузитесь |
| Приложение падает | Проверьте что `DEMO_MODE = true` (не false) |
