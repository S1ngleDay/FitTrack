## 🔍 Что проверить при оценивании?

### **Проверка 1: Включение демо (2 минуты)**

```bash
1. Откройте: src/utils/demoData.js
2. Проверьте: export const DEMO_MODE = true;  ✅
3. Перезагрузите приложение
4. Проверьте консоль: должно быть "✅ Loaded 20 demo workouts"
5. Откройте экран Статистика: должны быть графики
```

**Ожидаемый результат:** Приложение загружено с демо-данными ✅

### **Проверка 2: Отключение демо (1 минута)**

```bash
1. Измените: export const DEMO_MODE = false;
2. Перезагрузите приложение
3. Приложение работает, но демо-данные не загружаются
```

**Ожидаемый результат:** Чистое состояние приложения ✅

### **Проверка 3: Качество графиков**

```bash
1. Экран Статистика → за 28 дней
2. BarChart показывает 20 разных тренировок ✅
3. PieChart показывает распределение типов ✅
4. Числовые метрики правильные ✅
```

**Ожидаемый результат:** Красивые, информативные графики ✅

### **Проверка 4: БД данные (2 минуты)**

```bash
1. Экран Тренировки: список из 20 тренировок ✅
2. Кликнуть на силовую: есть упражнения ✅
3. Кликнуть на кардио: есть GPS маршрут и расстояние ✅
4. Проверить профиль: есть данные пользователя ✅
```

**Ожидаемый результат:** Все данные корректно отображаются ✅
