# ✅ Как использовать демо-режим FitTrack для презентации

## 🎯 Цель
Быстро сгенерировать реалистичные данные (20 тренировок, планы, профиль) для скрин-шотов презентации.

---

## 🚀 Быстрый старт (30 секунд)

### **Шаг 1: Включить демо-режим**

Откройте файл:
```
src/utils/demoData.js
```

Убедитесь, что в начале файла:
```javascript
export const DEMO_MODE = true;  // ✅ true = ВКЛЮЧЕН
```

### **Шаг 2: Перезагрузить приложение**

Полностью перезагрузите приложение (не горячий перезагруз):
- На Android/iOS эмуляторе: перезапустите приложение
- Или нажмите `r` в Expo CLI для полного перезагруза

### **Шаг 3: Проверить загрузку**

Откройте приложение и проверьте консоль:

```
🎮 Demo mode is ON - Initializing demo data...
✅ Loaded 20 demo workouts
✅ Loaded 3 demo plans
🎮 Demo data initialization complete!
```

### **Шаг 4: Готово! 🎉**

Теперь в приложении:
- ✅ На **главном экране** видны тренировки за сегодня
- ✅ На **экране Статистики** красивые графики
- ✅ На **экране Тренировок** список всех 20 тренировок
- ✅ **Профиль пользователя** заполнен (Петр Иванов, 75 кг...)

---

## 📊 Что видит пользователь?

| Экран | Данные |
|-------|--------|
| **Главная** | Сегодняшняя активность: 320 ккал, 12450 шагов |
| **Тренировки** | 20 тренировок за 3 месяца (бег, велосипед, силовая, кардио) |
| **Статистика** | Графики, диаграммы, распределение нагрузки |
| **Профиль** | Петр Иванов, 75 кг, 182 см, 28 лет |
| **Планы** | 3 готовых плана (марафон, мышцы, похудение) |

## 🎥 Гайд по скрин-шотам

### **Для красивых скрин-шотов статистики:**

1. Включите демо (см. выше)
2. Откройте экран **Статистика**
3. **Выберите период:** 28 дней (лучше всего смотрится)
4. Делайте скрин-шоты:
   - График активности (красивый BarChart)
   - Диаграмма типов тренировок (PieChart)
   - Нагрузка на мышцы (PieChart)
   - Сводка статистики (калории, время, расстояние)

### **Для скрин-шотов домашнего экрана:**

1. Выберите **период "7 дней"**
2. Виден список последних тренировок
3. Краткая статистика за день

### **Для скрин-шотов силовых упражнений:**

1. Откройте тренировку из списка типа **"Strength"**
2. Видны упражнения с весами, подходами, объемом
3. График прогресса (если есть несколько тренировок одного упражнения)

---

## 🛑 Выключение демо-режима

Когда закончите с презентацией, **обязательно выключите демо-режим:**

Откройте `src/utils/demoData.js` и измените:

```javascript
export const DEMO_MODE = false;  // ❌ false = ВЫКЛЮЧЕН
```

**Почему важно:**
- Демо-данные перезаписывают реальные данные пользователя
- Для релиза приложение должно работать без демо-режима
- Разработчики будут видеть чистое приложение без тестовых данных

## ⚡ Горячие клавиши (Expo CLI)

Во время разработки с демо-режимом:

- **`r`** — Полный перезагруз (рекомендуется после изменения демо-данных)
- **`w`** — Открыть приложение в браузере
- **`a`** — Открыть на Android эмуляторе
- **`i`** — Открыть на iOS симуляторе

## 🔍 Отладка

### **Демо-данные не загружаются?**

Проверьте:
1. ✅ `DEMO_MODE = true` в `src/utils/demoData.js`
2. ✅ Полный перезагруз приложения (не горячий)
3. ✅ Консоль показывает сообщения вроде "Loaded 20 demo workouts"

### **Хотите быстро добавить свои демо-данные?**

Отредактируйте массив `demoDemoWorkouts` в `src/utils/demoData.js`:
- Измените даты, метрики, упражнения
- Перезагрузите приложение
- Новые данные загрузятся автоматически
